# Movie Search

Movies list and search

Developed a responsive Movie Website using frontend technologies such as HTML, CSS, JavaScript and ReactJs. The website provides an attractive and user-friendly interface where users can browse various food recipes, view ingredients, cooking instructions, preparation time, and recipe images. The design focuses on easy navigation, mobile responsiveness, and an enhanced user experience. Features include recipe categories, search functionality, interactive recipe cards, and visually appealing layouts to help users quickly find and explore their favorite dishes.
